# Trailer Plan - Click the Brainrot! Roblox Game

Use real gameplay footage instead of AI-generated fake gameplay. This keeps the portfolio honest and shows the actual Roblox game.

## Recommended Trailer Length

30 to 45 seconds.

## Shot List

1. Title screen or Roblox game page
2. Player entering the game
3. First click/gameplay moment
4. Main loop shown clearly
5. Any upgrades, rewards, UI, shop, or progression
6. Fast montage of the best moments
7. End card with the Roblox game name and link

## Screenshot List

- Game thumbnail
- First in-game view
- Main gameplay moment
- UI/progression screen
- Any funny or high-energy moment

## Capture Options

- Windows Game Bar: press `Win + G`, then record gameplay
- OBS Studio: best option for clean trailer footage
- Roblox Studio screenshots: useful for clean portfolio images

## Notes

Keep private source files, place files, admin panels, and internal development tools out of screenshots and videos.
