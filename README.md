# Click the Brainrot! - Roblox Game Portfolio

Public portfolio page for **Click the Brainrot!**, a Roblox game built in Roblox Studio.

This repository is a showcase for the project, not a source-code release. The private development backup is kept separately.

## Project Overview

**Click the Brainrot!** is a Roblox clicker-style experience focused on fast interaction, simple progression, and bright, attention-grabbing presentation.

## What I Built

- Roblox Studio experience setup
- Clicker gameplay loop
- Public Roblox experience release
- Game thumbnail/branding
- Iteration-ready backup workflow using Git and GitHub
